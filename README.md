# AcademicChatterBot

A twitter bot that retweets the following hashtags [in order of search preference] based on Twitter's API rate limits:

-phdchat
-acwri 
-ecrchat 
-phdforum 
-phdadvice
-phdlife
-phdweekend
-gradschool
-phdgang
-phdhelp
-planningyourphd

The bot uses the Python library 'Tweepy' to interact with the Twitter API.
